Function Documentation

static SFLOAT cgmGQuality ( SFLOAT  input)
static

The function to determine the glucose measurement quality and set the value in the corresponding output variable.

The exact decision of the quality is implementation specific. For demonstration purpose, the quality is hardcoded to be 90%. Any other calculation algorithm should be implemented here.

Parameters
[in]input- the input glucose concentration
Returns
The quality value.
